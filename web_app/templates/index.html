Project Workflow

we noticed that many people became skeptical about the vaccines from Pfizer to Covaxin. Online discussions always seemed so one-sided, with many people misinformed about the vaccine. In addition, we saw that many anti-vaccine groups grew online, and we wanted to explore the trends associated with this. So, we decided to build a model that could aggregate tweets and analyze the sentiment trends.

Data Exploration

After scraping the data, we tried three approaches:

Supervised Learning:
  • Linear models: Logistic Regression
  • Non-linear Models: Naive Bayes, SVM, Decision Trees
    • Unsupervised Learning:
  • SVD + Topic Modelling
    • Deep Learning Models:
  • Baseline: RNN
  • More complex sequence models: GRU, LSTM (bidirectional)
    • model After training the accuracy for Deep learning is as high as 88% and for classifier it has the highest validation score of 76%
